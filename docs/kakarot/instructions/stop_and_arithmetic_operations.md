



# Stop and arithmetic operations

## Internal

### `internal_exp`
  
Internal exponentiation of two 256-bit integers from the stack.
#### Implicit arguments
  
```python  
range_check_ptr  
```
#### Explicit arguments
  
```python  
a (Uint256): The base.  
b (Uint256): The exponent.  
```
#### Returns
  
```python  
Uint256  
```
